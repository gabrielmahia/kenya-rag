# Contributing to kenya-rag

Thank you for your interest in contributing to kenya-rag!

## How to contribute

This repository accepts **bug reports** and **feedback** via GitHub Issues.

For feature requests or questions, please [open an issue](../../issues).

## Attribution

If you use kenya-rag in research, publications, or tools, please cite:


must align with that mission.
